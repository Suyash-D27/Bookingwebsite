# Sikkim Tourism App

## Getting Started

To run the application (both Backend and Frontend), open a terminal in the root directory and run:

```bash
npm run dev
```

This will start:
- **Backend** at `http://localhost:5000`
- **Frontend** at `http://localhost:5173` (or similar)

## Directory Structure
- `scr/`: Backend code (Node.js/Express)
- `frontend/`: Frontend code (React/Vite)

## Credentials
- **Backend Port**: 5000
- **Database**: MongoDB (Local)
